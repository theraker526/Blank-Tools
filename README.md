# blank tools

a curated collection of open source scripts for modular use

this repository is built on a singular purpose, all scripts are entirely free of hidden malware every tool script and module in this collection has been stripped to its core functionality for your own deployments

## overview

blank tools provides a versatile script kit and these scripts are designed to be dropped into any environment and it will work

* pure code no obfuscated payloads no backdoors no telemetry
* high performance optimized for efficiency and stealth where it can
* white labeled this collection is designed to be yours once cloned you are the owner

## features

* modular python scripts are designed to be standalone or integrated into larger frameworks
* minimizing dependencies installed
* transparent execution so what you see in the source is exactly what executes on the machine

* rats
